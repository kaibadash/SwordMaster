# SwordMaster

スタイリッシュソードアクションゲーム！

タップ操作で、剣を自在に操り

ダークマター四天王を　撃て！討て！

圧倒的爽快感がウリの新感覚Swordシューティングゲーム！

Sword Master ゆうすけの謎とは、いったい何なのか！？

[遊び方]

1. タップすると剣が発射されます

2. 画面を傾けると、自機が左右に動きます

3. 長押しタップすると剣のタメ撃ちができます

4. スワイプすると剣の発射角度が変わります
